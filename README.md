Task Tracker Web App

A simple and interactive web-based Task Tracker that helps you manage your daily tasks efficiently. This project showcases my skills in building user-friendly web applications using HTML, CSS, and JavaScript.


Features

Add tasks – Quickly add new tasks to your list.

Mark as complete – Keep track of completed tasks.

Delete tasks – Remove tasks once they are done.

Responsive design – Works on both desktop and mobile devices.


Technologies Used - HTML, CSS, and JavaScript
